Development tools for Your Data Health – automation, monitors, and utilities.

## Tools

### [Privacy Law Monitor](privacy-law-monitor/)

Automated weekly RSS monitor for state and international privacy law updates. Tracks IAPP, JD Supra, State AG Blog, and Privacy Daily—sends digest when new items are detected.

- **Schedule:** Every Wednesday at 00:00 UTC
- **Output:** Email digest via Resend
- **Open-source ready:** See [PUBLISHING.md](privacy-law-monitor/PUBLISHING.md) for hybrid (public tool + private service) approach
